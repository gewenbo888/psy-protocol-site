# Psy Protocol · Open Source Stack

An independent, bilingual (EN / 中文) field guide to the public open-source repositories under [github.com/PsyProtocol](https://github.com/PsyProtocol) and [github.com/QEDProtocol](https://github.com/QEDProtocol).

## Sections

1. **Core Protocol** — psy-node, psy-prover, psy-compiler, psy-sdk, psyup
2. **Dogecoin × Cross-chain** — psy-doge-solana-bridge, doge-on-solana, city-rollup, doge-light-client, doge-sdk, electrs-doge, txindex
3. **ZK Acceleration** — plonky2-hwa, plonky2-metal-demo, plonky2-wasm, plonky2.5, plonky2-toolbelt
4. **Cryptographic Primitives** — poseidon2, poseidon2-webgpu-demo, poseidon-goldilocks, noble-secp256k1, succinctx, winterfell, ark-swiftness, plonky3-fibonacci

Private components (psy-wallet, psy-blockchain-explorer, psy-services, parth-generic) are listed in the architecture diagram with a `(private)` marker but not linked.
